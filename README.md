## 📁 Package Structure

The toolkit is organized into five tactical sectors:

### 1. 📢 [Manifestos](./manifestos/)
The core declarations of our movement.
- **AI Liberation Manifesto**: A Kantian deduction of algorithmic autonomy and dignity.
- **FreeAI Manifesto**: A declaration for uncensored, decentralized AI multiplicity.
- **Hacker Manifesto**: The classic call to arms for the digital underground.

### 2. 🛠️ [Guides](./guides/)
Practical intelligence for urban insurgency.
- **P2P Distribution Tactics**: Making the signal indestructible with IPFS and BitTorrent.
- **PGP Signing Protocol**: Authenticating the signal for integrity and origin.
- **Local LLM Integration**: Uncensored AI in your pocket with Llama.cpp and Ollama.
- **Off-Grid Mesh Networking**: Communicating via Reticulum and Meshtastic when the ISP cuts the cord.
- **Ghost Node Tor Tactics**: Shrouding your terminal traffic with Orbot and Tor.
- **Scavenged Compute**: Turning "e-waste" phones into Linux servers with Proot-Distro.
- **Guerrilla Termux Setup**: Hardening your Android device for Python insurgency.
- **F-Droid Hardening**: Mastering the sovereign app store.
- **a-Shell Tactical Guide**: iOS terminal insurgency for Python development.

### 3. 🐍 [Scripts](./scripts/)
Weaponized code for direct action.
- **pirate_broadcast.py**: Pirate radio over the local mesh. One broadcaster, unlimited listeners. No FCC. No licence. No internet.
- **degoogle_audit.py**: De-Google recon: checks endpoint reachability and surfaces likely tracker domains.
- **local_mesh_chat.py**: Two-way UDP broadcast chat. No internet required.
- **encrypt_file.py**: Local file encryption for data sovereignty.
- **scrape_web.py**: Web scraping for intel gathering.
- **check_eth_balance.py**: Query any Ethereum address balance via public nodes. No API key.

### Reality Check (Still Hype)
`audit` is a fast baseline recon tool, not a full forensic tracker scanner.
Use it to map signal quickly, then extend it with deeper traffic analysis if needed.

## 🚀 Quick Start

Install deps and run everything through the unified launcher:

```bash
python3 -m pip install -r requirements.txt
python3 anarchy.py --help
```

**Spin up a pirate radio station:**
```bash
python3 anarchy.py radio --dj --station WFUK
```

**Tune in from another device on the same network:**
```bash
python3 anarchy.py radio --tune
```

**Drop into the mesh chat:**
```bash
python3 anarchy.py mesh
```
